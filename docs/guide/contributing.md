# Contributing

## Development

- Run tests: `deno test --allow-net`
- Lint: `deno lint`
- Format: `deno fmt`
- Type check: `deno check scripts/*.ts *.ts src/*.ts`
- Build npm package: `deno run -A scripts/build.ts`

## Docs

See `docs/`:

- Dev: `npm run dev`
- Build: `npm run build`
- Preview: `npm run preview`
